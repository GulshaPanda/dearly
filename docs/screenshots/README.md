# Screenshots

Drop the following PNGs into this folder. The repo README references them.

| Filename | What to capture |
|---|---|
| `landing.png` | Full landing page hero with the Group 10 collage. Take it on desktop at 1440px width. |
| `chat.png` | A conversation in progress — message bubbles + product cards visible. |
| `dashboard.png` | Welcome dashboard with two event cards in the bar and the "Let's create something meaningful" hero. |
| `paywall.png` | The trial paywall modal over a blurred chat background. |

## How to take great screenshots

1. Open the live site at https://dearly-beryl.vercel.app in Chrome.
2. Set browser width to 1440px (DevTools → Toggle device toolbar → Responsive → 1440).
3. Use the macOS screenshot tool (Cmd+Shift+4) or Windows Snipping Tool.
4. Save as `.png` at 2x retina resolution if possible.
5. Optionally annotate with arrows / labels in [Figma](https://figma.com) or [Excalidraw](https://excalidraw.com) — recruiters love a clean callout.

## Optional: animated GIF demo

A 5–10s GIF of the full flow (type message → see recommendation → click product) at the top of the README is 10x more eye-catching than static screenshots.

Tools to make one:
- macOS: [Kap](https://getkap.co) — record region → export as GIF
- Windows: [ScreenToGif](https://www.screentogif.com)
- Cross-platform: [Cleanshot X](https://cleanshot.com) (paid, exceptional)

Target file size <5 MB so GitHub renders it inline.
